### Commands
```
about - About
net - Who is online?
photo - Take the photo
screen - Take the screenshot
services - Get list of services
start - Start
stop - Stop
system - Get system information
torrent - List torrents
tv28 - Get TV28 remote
wake-pc - PC wakes up
```
